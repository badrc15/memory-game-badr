Memory Game & Vanishing Tic Tac Toe

This project contains two games:
Memory Game
Vanishing Tic Tac Toe

Memory Game:
Difficulty selection (Easy, Medium, Hard)
Randomised card decks (Animals, Numbers, Emoji)
Timer-based gameplay (lose condition)
Score tracking

Vanishing Tic Tac Toe:
Tic Tac Toe with a twist
Each player can only have 3 pieces on the board
When placing a 4th piece, the oldest piece disappears
Highlighted tile shows which piece will vanish next
Normal win condition still applies - 3 in a row

The project follows the Model-View-Controller pattern:

Model - Game logic
View - JavaFX UI
Controller - Handles user interaction and updates model/view

How to Run:
Have Java 17+ and Maven installed

Run MemoryGameApp.java from your IDE.
